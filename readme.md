fase 1: Inisiasi & Setup Database (Hari 1)Fase ini berfokus pada kerangka dasar backend dan manajemen "Smart Contract" (yang untuk versi prototipe ini akan kita simulasikan menggunakan relasi database yang ketat).Tech Stack: Python (FastAPI), SQLModel/SQLAlchemy (ORM), MySQL.Tugas Copilot: Kamu bisa langsung prompt Copilot untuk membuat boilerplate FastAPI lengkap dengan konfigurasi koneksi MySQL.Skema Database Inti: Minta Copilot membuatkan model tabel untuk:Users (Role: UMKM, Streamer, Clipper).Donations (Mencatat nominal, timestamp, pesan).Clip_Tasks (Mewakili instruksi otomatis ke clipper).IP_Licenses (Simulasi smart contract: mencatat UUID Streamer yang memberikan izin ke UUID Clipper).Fase 2: Mesin Deteksi & Trigger Otomatis (Hari 2-3)Ini adalah nyawa dari arsitektur Catalyst yang kamu rancang. Kita akan mengubah simulasi terminal POC-mu menjadi API yang sesungguhnya.Endpoint Donasi (/api/donations/stream): Endpoint yang menerima payload JSON (pesan chat dan nominal) secara real-time.Logika NLP Sederhana (Keyword Extraction): Daripada melatih model AI dari awal, gunakan Copilot untuk menulis fungsi Python berbasis RegEx atau library NLTK sederhana untuk mendeteksi kata kunci UMKM (misal: "produk", "beli", "keripik") dari pesan donasi.Timestamp Generator: Fungsi yang secara otomatis menghitung start_time dan end_time (misal: 1 menit sebelum dan 2 menit sesudah donasi masuk) untuk diberikan ke clipper.Fase 3: Clipper Dashboard API (Hari 4)Fase ini menyiapkan data yang nantinya akan dikonsumsi oleh frontend (atau sekadar diuji via Postman).Endpoint Antrean (/api/clips/queue): Membuat API yang mengembalikan data JSON berisi daftar tugas potong video untuk clipper tertentu, lengkap dengan parameter timestamp dan URL raw footage.Sistem Validasi Akses: Fungsi untuk memastikan hanya clipper dengan token/lisensi aktif yang bisa menarik data raw footage tersebut.Fase 4: Simulasi "Micro-Endorsement" & Pengujian (Hari 5)Menggabungkan seluruh alur kerja dari input hingga output.Automation Workflow: Menggunakan Copilot untuk menyambungkan event donasi agar otomatis melakukan insert ke tabel Clip_Tasks tanpa jeda manual.Testing: Menjalankan script Python untuk "menembak" endpoint donasi dengan dummy data dan memastikan clipper langsung menerima tugasnya secara instan.
